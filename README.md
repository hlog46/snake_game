# 贪吃蛇游戏 Snake Game

经典贪吃蛇游戏的现代化复刻版，基于 Python + Pygame 开发，支持多种难度、多种视觉主题与丰富的打击特效。

## 功能特性

- **三档难度**：简单 / 普通 / 困难（含障碍物、食物倒计时）
- **三种主题**：经典像素风 / 霓虹赛博风 / 自然森林风
- **丰富特效**：粒子爆炸、连击 Combo、死亡碎裂、屏幕闪光
- **本地存档**：最高分持久化存储

## 安装与运行

```bash
pip install pygame
python main.py
```

## 操作方式

| 按键 | 功能 |
|------|------|
| ↑↓←→ / WASD | 控制方向 |
| ESC / P | 暂停 |

## 项目结构

```
snake_game/
├── main.py          # 入口
├── config/          # 配置（设置、字体）
├── game/            # 游戏逻辑（蛇、食物、棋盘、状态机）
├── effects/         # 特效（粒子、连击、死亡、分数弹出）
├── themes/          # 视觉主题
├── ui/              # 界面（菜单、HUD）
└── data/            # 本地存档
```

## 依赖

- Python 3.x
- pygame

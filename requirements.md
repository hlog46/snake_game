# 贪吃蛇游戏 —— 需求文档（PRD）

**版本：** 1.0
**日期：** 2026-03-24
**状态：** 草稿

---

## 1. 项目概述

### 1.1 背景
本项目为一款经典贪吃蛇游戏的现代化复刻版，在保留核心玩法的基础上，加入难度分级、多种画面风格与丰富的打击特效，提升玩家的沉浸感与可玩性。

### 1.2 目标用户
- 休闲玩家：寻求轻松娱乐体验
- 竞技玩家：挑战更高分数与困难模式
- 怀旧玩家：享受经典玩法的同时体验视觉升级

### 1.3 核心玩法
玩家控制一条不断移动的蛇，通过吃食物使蛇身增长并积累分数。蛇碰到墙壁或自身则游戏结束。游戏通过难度、主题和特效三个维度提供差异化体验。

## 2. 功能需求

### 2.1 核心游戏机制

#### 2.1.1 蛇的移动
- 蛇以固定频率向当前方向前进，每次移动一格
- 玩家通过方向键（↑↓←→）或 WASD 控制蛇的转向
- 不允许 180° 掉头（如正在向右移动，不能直接向左）
- 移动速度由当前难度决定（见 2.2 节）

#### 2.1.2 蛇身增长
- 蛇吃到食物后，尾部增加一节，蛇身长度 +1
- 吃到特殊食物（加速道具/减速道具）时，额外增长 2 节

#### 2.1.3 食物生成
- 普通食物：随机生成在空白格子，始终存在一个
- 特殊食物：每隔 20 秒随机生成一个，持续 8 秒后消失
  - 加速道具（红色闪光）：蛇速暂时提高 50%，持续 5 秒，得分 ×2
  - 减速道具（蓝色冰晶）：蛇速暂时降低 30%，持续 5 秒，得分 ×1.5
- 困难模式下，普通食物在 **食物消失倒计时**（见 2.2 节）到期后刷新位置

#### 2.1.4 碰撞检测
| 碰撞类型 | 结果 |
|---------|------|
| 蛇头碰到墙壁 | 游戏结束，触发死亡特效 |
| 蛇头碰到自身蛇身 | 游戏结束，触发死亡特效 |
| 蛇头碰到障碍物（困难模式） | 游戏结束，触发死亡特效 |
| 蛇头碰到普通食物 | 吃食物，触发吃食特效 |
| 蛇头碰到特殊食物 | 吃食物，触发特殊特效 |

#### 2.1.5 分数计算
- 普通食物：+10 分
- 特殊食物（加速道具）：+30 分，激活期间每个普通食物得分 ×2
- 特殊食物（减速道具）：+20 分，激活期间每个普通食物得分 ×1.5
- 连击加成：连续在 3 秒内吃到食物，触发 Combo，每连击一次额外 +5 分
- 最高分本地持久化存储

---

### 2.2 难度调整系统

提供三档难度，游戏开始前选择，游戏中不可切换。

| 参数 | 简单（Easy） | 普通（Normal） | 困难（Hard） |
|------|-------------|--------------|-------------|
| 蛇移动速度（格/秒） | 8 | 12 | 18 |
| 蛇初始长度 | 3 节 | 3 节 | 5 节 |
| 食物消失倒计时 | 无 | 无 | 6 秒 |
| 障碍物数量 | 0 | 0 | 5 个随机障碍 |
| 分数倍率 | ×1 | ×1.5 | ×2.5 |
| 特殊食物出现间隔 | 30 秒 | 20 秒 | 15 秒 |

**说明：**
- 困难模式下，5 个障碍物在游戏开始时随机分布于棋盘，避开蛇初始位置
- 困难模式下，食物倒计时通过 HUD 上的进度条显示

---

### 2.3 画面风格调整系统

提供三种视觉主题，可在主菜单或暂停菜单中随时切换，立即生效。

#### 主题一：经典像素风（Classic Pixel）
- 背景：深绿色 `#1a2e1a`
- 蛇身：亮绿色方块，头部用深绿色标识
- 食物：红色方块
- 障碍物：灰色方块
- 特效颜色：黄绿色粒子
- 字体：像素风等宽字体

#### 主题二：霓虹赛博风（Neon Cyber）
- 背景：近黑色 `#0d0d1a`
- 蛇身：带发光轮廓的青色/紫色渐变（蛇头到蛇尾颜色渐变）
- 食物：粉红色/洋红色发光圆形
- 障碍物：深紫色矩形带霓虹边框
- 特效颜色：青色/品红色粒子
- 字体：现代无衬线字体，带发光效果

#### 主题三：自然森林风（Forest Nature）
- 背景：深棕色土地纹理 `#2d1f0e`
- 蛇身：渐变绿色圆角矩形（模拟鳞片）
- 食物：橙红色苹果/浆果图案
- 障碍物：棕色树桩图案
- 特效颜色：金黄色/橙色粒子（落叶效果）
- 字体：圆润可爱风格字体

---

### 2.4 打击特效系统

#### 2.4.1 吃食物特效
- **粒子爆炸**：在食物位置生成 12~20 个粒子，向四周扩散，0.5 秒内消散
- **分数跳出动画**：得分数字从食物位置向上浮动，1 秒内淡出消失
- **蛇头闪光**：蛇头在吃食时短暂（0.1 秒）放大 1.2 倍并亮起

#### 2.4.2 吃特殊食物特效
- **全屏光晕**：吃到特殊食物时，屏幕边缘出现 0.3 秒的彩色光晕
- **道具图标动画**：HUD 区域的道具状态图标出现旋转进场动画
- **蛇身渲染变化**：激活期间蛇身颜色/发光效果发生变化以提示状态

#### 2.4.3 连击特效（Combo）
- 触发条件：在 3 秒内连续吃到 3 个及以上食物
- **Combo 文字**：屏幕中央出现"COMBO ×N"大字，颜色随连击数加深，0.8 秒后消失
- **震动特效**：屏幕轻微震动（±3px，持续 0.2 秒）
- **粒子增强**：Combo 状态下粒子数量 +50%，粒子速度 +30%

#### 2.4.4 死亡特效
- **蛇身碎裂动画**：蛇身每节变成独立碎片，向外飞散（速度随机，0.8 秒消失）
- **屏幕红闪**：全屏红色半透明遮罩，0.3 秒内从 60% 透明度衰减到 0
- **慢动作效果**：死亡瞬间游戏时间短暂（0.2 秒）减速到 20%，增强打击感
- **粒子残留**：蛇头位置生成大量粒子，颜色与当前主题主色调一致

### 2.5 UI 与菜单

#### 2.5.1 主菜单
```
[游戏 LOGO / 标题动画]

  ▶  开始游戏
  ⚙  选择难度       → [简单 / 普通 / 困难]
  🎨  选择风格       → [经典像素风 / 霓虹赛博风 / 自然森林风]
  🏆  排行榜         → [显示本地 Top 10 最高分]
  ✕  退出游戏
```
- 选中项有高亮/光标动画
- 背景有循环的蛇身移动装饰动画

#### 2.5.2 游戏内 HUD
- 左上角：当前分数（大字）、最高分（小字）
- 右上角：当前难度标签、当前主题名
- 下方（困难模式）：食物倒计时进度条
- 道具激活时：屏幕上方中央显示剩余时间进度条 + 道具图标

#### 2.5.3 暂停菜单
- 按 ESC 或 P 键触发
- 选项：继续游戏 / 重新开始 / 更换风格 / 返回主菜单
- 背景游戏画面模糊处理

#### 2.5.4 游戏结束界面
- 显示本次分数（带数字跳动动画）
- 与最高分对比：若刷新最高分显示"NEW RECORD!"金色特效
- 选项：再来一局 / 返回主菜单

## 3. 非功能需求

| 指标 | 要求 |
|------|------|
| 目标平台 | Windows / macOS / Linux 桌面端 |
| 帧率 | 目标 60FPS，最低 30FPS |
| 键盘响应延迟 | < 16ms（一帧内响应） |
| 首次启动时间 | < 3 秒 |
| 数据持久化 | 最高分本地存储（JSON 文件） |
| 分辨率支持 | 默认 800×600，支持 1024×768 |

---

## 4. 不在范围内（Out of Scope）

- 联网对战 / 排行榜
- 移动端适配
- 关卡设计（当前为无尽模式）
- 音效系统（留作后续迭代）

---

## 5. 验收标准

1. 三档难度可在主菜单正确切换，游戏速度/障碍物/食物倒计时与规格一致
2. 三种主题可在主菜单和暂停菜单切换，切换立即生效
3. 吃食物时粒子特效和分数浮动动画正常播放
4. 连续吃 3 个食物内出现 Combo 文字和屏幕震动
5. 死亡时蛇身碎裂动画和红色屏幕闪光正常播放
6. 最高分在关闭游戏后重新打开仍能保留
